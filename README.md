Este é um projeto com o intuito de usar queries de SQL para realizar os processos de limpeza e formatação de uma base de dados habitacionais.

Apesar de constituir etapa prévia à análise dos dados em si, é de fundamental importância manter uma base de dados organizada e limpa para otimização dos trabalhos posteriores.

Durante o processo realizamos operações como a retirada de valores nulos, formatações, tabulações dos dados e remoção de duplicatas.
